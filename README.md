# 哔哩哔哩商业化全景

一个单文件、交互式的可视化页面，梳理 B站（哔哩哔哩）2025 全年的商业化版图。

## 内容

- **四大业务板块**：广告 / 增值服务 / 游戏 / 电商 & IP —— 点击任意板块弹出详情，含「核心介绍 · 业务本质」「用户画像 × 商业需求」「营收来源」「增长/下降归因」。
- **商业化基础设施**：哔哩必达、MATES 人群模型、风控、创作者激励等，点击 ↗ 跳转对应官网。
- **用户画像 · 社区底盘**：MAU / DAU / 时长 / 年龄等核心 KPI、人口结构、兴趣圈层与用户特质。
- 每个模块右上角 ↗ 链接到对应的 B站官方服务。

## 使用

直接用浏览器打开 [`index.html`](index.html) 即可，无需构建、无外部依赖（CSS / JS 全部内联）。

> 数据来源：B站 2025 年公开财报及第三方人群口径，仅供学习与展示用途。
